# Паттерны и привычные времена (шаблон)

Скопируй этот файл в свой Obsidian vault: `.claude/day-patterns.md`

Claude использует его при планировании дня.

## Утренняя рутина (до планирования)

Что обычно делается ДО запуска `/day:today`:

- Завтрак
- Медитация
- ...

**Планирование начинается с ??:00.**

## Буферы

- Между deep work блоками: 5-10 мин
- Перед важными встречами: 10-15 мин
- После встреч: 5-10 мин
