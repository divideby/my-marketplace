# Паттерны и привычные времена

Этот файл содержит типичные времена для повторяющихся активностей. Claude использует его при планировании дня.

## Утренняя рутина (до планирования)

Эти активности обычно выполняются ДО запуска `/day:today`:

| Активность | Типичное время | Длительность |
|------------|----------------|--------------|
| Завтрак | 07:00 | 30 мин |
| Медитация | 07:30 | 15 мин |
| Дневник | 07:45 | 15 мин |
| Чтение | 08:00 | 30-60 мин |
| Самообразование | 08:30 | 60-90 мин |

**Примечание**: Планирование обычно начинается после утренней рутины (~10:00).

## Фиксированные регулярные события

События, которые НЕЛЬЗЯ перенести и происходят в одно время:

| Событие | Типичное время | День недели | Примечание |
|---------|----------------|-------------|------------|
| Оплата квартиры | 19:00 | по дате | ~12 числа |
| Тренировка | 17:00-18:30 | пн/ср/пт | |

## Предпочтения по времени

| Тип активности | Лучшее время | Причина |
|----------------|--------------|---------|
| Deep work | 10:00-13:00 | Максимальная концентрация |
| Встречи | 14:00-17:00 | После обеда |
| Админ задачи | 16:00-17:00 | Низкая энергия |
| Обед | 13:00-14:00 | |

## Буферы

- Между deep work блоками: 5-10 мин
- Перед важными встречами: 10-15 мин на подготовку
- После встреч: 5-10 мин на заметки
