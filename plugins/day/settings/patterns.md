# Паттерны и привычные времена

Этот файл содержит типичные времена для повторяющихся активностей. Claude использует его при планировании дня.

## Утренняя рутина (до 10:00)

Всё это выполняется ДО запуска `/day:today` и укладывается до 10:00:

- Завтрак
- Медитация
- Дневник
- Чтение
- Самообразование

**Планирование начинается с 10:00.**

|---------|----------------|-------|------------|

| Тренировка (выходной) | 09:00 | сб | |

## Временные слоты

| Время | Особенность |
|-------|-------------|
| 10:00-12:00 | Свободно от встреч, подходит для deep work |
| 13:00-14:00 | Обед |

## Буферы

- Между deep work блоками: 5-10 мин
- Перед важными встречами: 10-15 мин на подготовку
- После встреч: 5-10 мин на заметки
