# books

Пайплайн чтения книг по методологии Поварнина.

## Назначение

Система для осознанного чтения книг с фиксацией прогресса, рефлексией и извлечением ключевых идей. Основана на методологии С.И. Поварнина ("Как читать книги").

## Структура

```
books/
├── .claude-plugin/
│   └── plugin.json
├── commands/
│   ├── new.md        # /books:new — добавить книгу
│   ├── session.md    # /books:session — сессия чтения
│   ├── progress.md   # /books:progress — статистика
│   └── finish.md     # /books:finish — завершить книгу
└── skills/
    └── extracting-book-toc/
        ├── SKILL.md
        └── scripts/
            └── fetch-toc.py
```

## Команды

| Команда | Описание |
| ------- | -------- |
| `/books:new` | Добавить новую книгу в библиотеку |
| `/books:session` | Начать сессию чтения |
| `/books:progress` | Показать прогресс по книгам |
| `/books:finish` | Завершить книгу с рефлексией |

## Skills

| Skill | Триггер |
| ----- | ------- |
| `extracting-book-toc` | Запрос оглавления книги, списка глав |

## Хранение данных

- **Заметки книг:** `Base/<Название книги>.md`
- **Сессии чтения:** `Base/Сессия-<Книга>-YYYY-MM-DD.md`

## Workflow чтения

1. **Добавить книгу** (`/books:new`)
   - Найти на LitRes, получить ID
   - Определить цель и способ чтения
   - Создать заметку книги

2. **Извлечь оглавление** (skill автоматически)
   - Скрипт fetch-toc.py: LitRes, Open Library, Google Books
   - Fallback: ручной ввод

3. **Сессии чтения** (`/books:session`)
   - Вспомнить контекст
   - Читать (мин. 10 минут)
   - Рефлексия по Поварнину
   - Обновить прогресс

4. **Завершение** (`/books:finish`)
   - Итоговая рефлексия
   - Ключевые идеи
   - Применение

## Способы чтения

- **Активное** — для самообразования: оценка, критика, выводы
- **Пассивное** — для художественных произведений: погружение

## Источники метаданных

| Источник | Использование |
| -------- | ------------- |
| LitRes | Русскоязычные книги (рекомендуется) |
| Open Library | Англоязычные, ISBN |
| Google Books | Fallback |
