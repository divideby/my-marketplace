# Новая книга

Добавь новую книгу для чтения по методологии Поварнина.

## Фаза 0: Выбор книги

### Шаг 1: Найти книгу на ЛитРес

Спроси у пользователя название книги. Попроси найти её на litres.ru и скопировать ID из URL:

```
litres.ru/book/author/title-48514275/ → ID = 48514275
```

### Шаг 2: Получить метаданные

```bash
# Русскоязычные книги — ЛитРес (рекомендуется)
python3 plugins/books/skills/extracting-book-toc/scripts/fetch-toc.py --litres-id "48514275" --info

# Англоязычные книги — Google Books
python3 plugins/books/skills/extracting-book-toc/scripts/fetch-toc.py --title "Deep Work" --info
```

Скрипт выведет YAML frontmatter с метаданными: название, автор, обложка, страницы, ISBN.

### Шаг 3: Определить цель чтения

Спроси пользователя:

**Зачем читаешь эту книгу?**

- Самообразование — глубокое изучение темы
- Произведение искусства — эстетическое удовольствие
- Осведомление — получить общее представление
- Воодушевление — мотивация и вдохновение

### Шаг 4: Выбрать способ чтения

На основе цели предложи способ:

- **Активное чтение** — для самообразования, осведомления. Оценка, критика, выводы.
- **Пассивное чтение** — для произведений искусства (первое прочтение). Погружение без критики.

### Шаг 5: Сформировать ожидания

Спроси:

- О чём, по-твоему, эта книга?
- Какие вопросы хочешь решить с её помощью?

## Шаблон заметки книги

Создай файл `Base/Название книги.md`:

```markdown
---
title: "Название"
author: [Автор]
publisher: Издательство
publish: YYYY
total: N
isbn: XXXXX
coverUrl: URL обложки
litresId: ID
status: reading
created: YYYY-MM-DD HH:MM:SS
updated: YYYY-MM-DD HH:MM:SS
tags: [own]
---

![cover|150](URL обложки)

[[00 Books]]

## Цель чтения

(Зачем читаю)

## Способ чтения

(Активное / Пассивное)

## Ожидания

(Что хочу получить)

## Прогресс

- [ ] Предисловие/Введение
- [ ] Глава 1
- [ ] ...

## Ключевые идеи

(Заполняется по ходу и в конце)

## Вопросы и мысли

(Заполняется по ходу чтения)

## Итог

(Заполняется после прочтения)
```

## Следующий шаг

После создания заметки предложи:

1. Извлечь оглавление книги (skill `extracting-book-toc` активируется автоматически)
2. Начать первую сессию с `/books:session`
