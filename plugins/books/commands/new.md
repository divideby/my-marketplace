# Новая книга

Добавь новую книгу для чтения по методологии Поварнина.

## Фаза 0: Выбор книги

### Шаг 1: Найти книгу

Спроси у пользователя название книги или ISBN. Затем найди информацию через Google Books API:

```bash
curl -s "https://www.googleapis.com/books/v1/volumes?q=ISBN:XXXXX" | jq '.items[0].volumeInfo'
```

Извлеки: title, authors, publisher, publishedDate, pageCount, imageLinks.thumbnail

### Шаг 2: Определить цель чтения

Спроси пользователя:

**Зачем читаешь эту книгу?**
- Самообразование — глубокое изучение темы
- Произведение искусства — эстетическое удовольствие
- Осведомление — получить общее представление
- Воодушевление — мотивация и вдохновение

### Шаг 3: Выбрать способ чтения

На основе цели предложи способ:

- **Активное чтение** — для самообразования, осведомления. Оценка, критика, выводы.
- **Пассивное чтение** — для произведений искусства (первое прочтение). Погружение без критики.

### Шаг 4: Сформировать ожидания

Спроси:
- О чём, по-твоему, эта книга?
- Какие вопросы хочешь решить с её помощью?

## Шаблон заметки книги

Создай файл `Sources/Название книги.md`:

```markdown
---
title: "Название"
author: [Автор]
publisher: Издательство
publish: YYYY-MM-DD
total: N (страниц)
isbn: XXXXX
coverUrl: URL обложки
status: reading
created: YYYY-MM-DD HH:MM:SS
updated: YYYY-MM-DD HH:MM:SS
tags: [own]
---

![cover|150](URL обложки)

[[00 Books]]

## Цель чтения

(Зачем читаю)

## Способ чтения

(Активное / Пассивное)

## Ожидания

(Что хочу получить)

## Прогресс

- [ ] Предисловие/Введение
- [ ] Глава 1
- [ ] ...

## Ключевые идеи

(Заполняется по ходу и в конце)

## Вопросы и мысли

(Заполняется по ходу чтения)

## Итог

(Заполняется после прочтения)
```

## Следующий шаг

После создания заметки предложи:
1. Запустить `/books:toc` для извлечения оглавления
2. Начать первую сессию с `/books:session`
