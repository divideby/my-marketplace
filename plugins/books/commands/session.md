# Сессия чтения

Проведи сессию чтения по методологии Поварнина.

## Подготовка

### Шаг 1: Выбрать книгу

Спроси пользователя, какую книгу читаем. Если не указана — предложи список книг со статусом `reading`.

### Шаг 2: Вспомнить контекст

Прочитай заметку книги и найди:
- Где остановился (последний отмеченный пункт в Прогрессе)
- Последние записи в "Вопросы и мысли"
- Цель и способ чтения

Кратко напомни пользователю эту информацию.

## Сессия

### Шаг 3: Создать файл сессии

**ВАЖНО:** Обязательно укажи текущую дату И время начала сессии (`started`). Это нужно для отслеживания продолжительности чтения.

Создай файл `Base/Сессия-Название книги-YYYY-MM-DD.md`:

```markdown
---
tags:
  - reading-session
book: "[[Название книги]]"
date: YYYY-MM-DD
started: HH:MM
ended:
---

**Прочитано:** Глава X / страницы N-M

## Ключевые мысли

-

## Вопросы

-

## Связи

-
```

### Шаг 4: Читать

Скажи пользователю, что он может начинать читать. Минимальная длина сессии: 10 минут.

Напомни правила активного чтения (если выбран этот способ):
- Оценивай мысли автора
- Соглашайся или критикуй
- Делай выводы
- Связывай с тем, что уже знаешь

## Завершение сессии

**ВАЖНО:** Сессия чтения — это полный цикл от начала чтения до завершения обсуждения и фиксации результатов. Поле `ended` в файле сессии заполняется ТОЛЬКО после прохождения всех шагов ниже (обработка → рефлексия → обновление записей). Не завершай сессию преждевременно — дождись, пока пользователь расскажет о прочитанном и вы вместе всё обсудите.

### Шаг 5: Обработка

Когда пользователь сообщит, что закончил читать, проведи обсуждение. Спроси:
1. Какие главы/страницы прочитал?
2. Какие ключевые мысли запомнились?
3. Какие вопросы возникли?
4. С чем связалось из уже известного?

### Шаг 6: Рефлексия

Задай вопросы по Поварнину:
- Что понял?
- Что не понял?
- С чем согласен / не согласен?
- Как это связано с тем, что уже знаешь?

### Шаг 7: Обновить записи

1. Заполни файл сессии ответами пользователя
2. Обнови прогресс в заметке книги (отметь прочитанные главы `[x]`)
   - **Важно:** Сохраняй веса `[w:N]` при отметке — они нужны для точного расчёта прогресса
   - Пример: `- [x] Глава 1 [w:125]` (не удаляй `[w:125]`)
3. Добавь ключевые мысли в "Ключевые идеи"
4. Добавь вопросы в "Вопросы и мысли"
5. **Проставь `ended: HH:MM`** — текущее время окончания сессии (только сейчас, после завершения всего обсуждения)

## Следующий шаг

Предложи:

- Продолжить чтение позже с `/books:session`
- Посмотреть прогресс с `/books:progress`
- Если книга дочитана — `/books:finish`
