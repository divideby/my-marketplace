# specs

Плагин для создания и управления спецификациями проектов.

## Назначение

Автоматизация создания структурированных спецификаций для инженерных, исследовательских и управленческих задач. Два уровня детализации:

- **L0** — one-liner для быстрых задач
- **L1** — полная страница для задач с рисками/зависимостями

## Структура

```
specs/
├── .claude-plugin/
│   └── plugin.json
└── commands/
    └── new.md        # /specs:new
```

## Команды

| Команда | Описание |
| ------- | -------- |
| `/specs:new` | Создать новую спецификацию |

## Формат спецификаций

**Имя файла:** `S-YYYY-MM-DD-XXX_Title.md`

**Типы:** engineering, research, management

**Статусы:** idea → draft → review → active → done/paused/killed

## Шаблон L1

1. Контекст
2. Цель
3. Скоуп (goals / non-goals)
4. Требования
5. Интерфейсы
6. Acceptance criteria
7. Риски/допущения
8. План
