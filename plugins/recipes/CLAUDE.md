# Recipes Plugin

Плагин для управления рецептами в Obsidian.

## Команды

| Команда | Описание |
|---------|----------|
| `/recipes:new` | Добавить новый рецепт в базу знаний |
| `/recipes:suggest` | Подобрать рецепт по продуктам в холодильнике |

## Настройка

Создай файл `.claude/recipes.local.md` в проекте:

```markdown
---
vault_path: ~/Yandex.Disk/Ocean/new-ocean/Base
favorites:
  - курица
  - паста
  - чеснок
  - помидоры
dislikes:
  - кинза
  - острое
  - анчоусы
dietary: []
---

# Мои предпочтения в еде

Дополнительные заметки о предпочтениях.
```

### Поля настроек

| Поле | Описание | По умолчанию |
|------|----------|--------------|
| `vault_path` | Путь к папке с рецептами в Obsidian | `~/Yandex.Disk/Ocean/new-ocean/Base` |
| `favorites` | Любимые продукты (приоритет при подборе) | `[]` |
| `dislikes` | Нелюбимые продукты (исключаются из подбора) | `[]` |
| `dietary` | Диетические ограничения | `[]` |

## Формат рецептов

Рецепты хранятся как Markdown файлы с YAML frontmatter:

```markdown
---
tags:
  - recipe
  - ужин
ingredients:
  - курица
  - чеснок
  - оливковое масло
cook_time: "45 мин"
servings: 4
created: 2026-01-14
---

# Курица с чесноком

## Ингредиенты

- [ ] Курица — 500г
- [ ] Чеснок — 4 зубчика
- [ ] Оливковое масло — 2 ст.л.

## Приготовление

1. Разогреть духовку до 200°C
2. ...
```

Файлы именуются: `Рецепт - Название блюда.md`
