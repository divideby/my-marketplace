# Добавление нового рецепта

## Задача

Создать файл рецепта в Obsidian vault в формате Markdown с YAML frontmatter.

## Настройки

Прочитай настройки из `.claude/recipes.local.md` в текущем проекте. Если файла нет, используй значения по умолчанию:
- `vault_path`: `~/Yandex.Disk/Ocean/new-ocean/Base`

## Процесс

1. **Получи информацию о рецепте** от пользователя:
   - Название блюда
   - Ингредиенты (список)
   - Шаги приготовления
   - Время приготовления (опционально)
   - Категория: завтрак, обед, ужин, десерт, закуска (опционально)

2. **Сформируй файл** в формате:

```markdown
---
tags:
  - recipe
  - [категория]
ingredients:
  - ингредиент 1
  - ингредиент 2
cook_time: "30 мин"
servings: 2
created: YYYY-MM-DD
---

# Название блюда

## Ингредиенты

- [ ] Ингредиент 1 — количество
- [ ] Ингредиент 2 — количество

## Приготовление

1. Первый шаг
2. Второй шаг
3. ...

## Заметки

Дополнительные советы или вариации.
```

3. **Сохрани файл** в vault_path с именем `Рецепт - Название блюда.md`

4. **Подтверди** пользователю создание файла и покажи путь.

## Советы

- Если пользователь даёт ссылку на рецепт — используй WebFetch чтобы получить информацию
- Если пользователь описывает рецепт устно — структурируй информацию
- Используй чекбоксы `- [ ]` для ингредиентов — удобно отмечать при готовке
